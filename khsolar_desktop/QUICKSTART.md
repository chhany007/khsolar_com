## Keyboard Shortcuts

- `Ctrl+N` - New record (in current tab)
- `Ctrl+E` - Export data
- `Ctrl+F` - Focus search box
- `Ctrl+Q` - Quit application

## Database Location

All data is stored in SQLite databases in the `data/` folder:
- `data/products.db` - Product catalog
- `data/sales.db` - Sales records
- `data/warranty.db` - Warranty information
- `data/customers.db` - Customer database

## Backup

To backup your data, simply copy the entire `data/` folder to a safe location.
